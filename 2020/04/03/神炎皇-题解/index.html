神炎皇 题解

题面：

【问题描述】
神炎皇乌利亚很喜欢数对，他想找到神奇的数对。
对于一个整数对$(a,b)$，若满足$a+b \le n$且$a+b$是$ab$的因子，则成为神奇的数对。请问这样的数对共有多少呢？

【输入格式】
一行一个整数n。
【输出格式】
一行一个整数表示答案，保证不超过64位整数范围。
【样例输入输出】
uria.in
21
uria.out
11
【数据范围与约定】
对于$20\%$的数据$n \le 1000$；
对于$40\%$的数据$n \le 100000$；
对于$60\%$的数据$n \le 10000000$；
对于$80\%$的数据$n \le 1000000000000$；
对于$100\%$的数据$n \le 100000000000000$。

解析：

设 $ \gcd(a,b) = d $

令$a’=\frac{a}{d} , b’=\frac{b}{d}$

易得$da’=a,db’=b$且$\gcd(a’,b’)=1 $

代回得$d(a’+b’) \mid d^2a’b’$
所以$a’+b’ \mid da’b’$
又因为$\gcd(a’,b’)=1$
所以$\gcd(a’+b’,b’) = \gcd(a’,a’+b’) =1$
所以$a’+b’ \nmid a’b’$
又因为$a’+b’ \mid da’b’$
所以$a’+b’ \mid d$
因为$a+b \le n$
所以$d(a’+b’) \le n$
所以$a’+b’ \le \sqrt{n}$
设$a’+b’=c,d=ck$
因为$d(a’+b’) \le n$
所以$c^2k \le n$
所以$k \le \dfrac{n}{c^2}$
所以$k$有$\left\lfloor\dfrac{n}{c^2}\right\rfloor$个
因为$\gcd(a’+b’,b’) = \gcd(a’,a’+b’) =1$
所以$\gcd(a’,c)= \gcd(b’,c) =1$
所以数对$(a’,b’)$有$\varphi(c)$个
